# # Netflix Data Analysis Dashboard 📺

> **Tool:** Power BI · **Dataset:** Netflix Movies and TV Shows (Kaggle)  
> **Type:** Business Intelligence Dashboard

---

## 📌 Project Overview

Built an interactive multi-page Power BI dashboard to analyse the Netflix content library.  
The goal was to uncover content trends, production patterns, and genre distributions that a content strategy team could act on.

---

## 📊 Dataset

- **Source:** [Netflix Movies and TV Shows — Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- **Size:** 8,807 titles
- **Fields:** Title, Type (Movie/TV Show), Director, Country, Date Added, Release Year, Rating, Duration, Genre

---

## 🔍 Key Insights Uncovered

- **70% of all Netflix content was added after 2015** — reflecting the streaming boom era
- **Movies outnumber TV Shows** at roughly 70:30 ratio across the entire catalogue
- **United States, India, and United Kingdom** are the top 3 content-producing countries
- **Dramas and Comedies** dominate genre distribution across both movies and shows
- Content additions **peak in Q4 (Oct–Dec)** each year, likely aligned with subscriber growth targets

---

## 📐 Dashboard Pages

| Page | Description |
|------|-------------|
| Overview | Total titles, Movies vs TV Shows split, content added over years |
| Genre Analysis | Genre breakdown by type and country |
| Country Map | Geographic distribution of content production |
| Ratings | Rating distribution (TV-MA, PG-13, etc.) across content types |
